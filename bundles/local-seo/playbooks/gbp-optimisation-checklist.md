A short, high-leverage checklist for the first pass on a new local client. Pair it with the full [local SEO dimensions](/local-seo/) for the deep audit.

# Quick wins

1. Claim and optimise **Apple Business Connect** — adoption is low (~16% per BrightLocal), so it's an easy win.
2. Claim and optimise **Bing Places** — the Bing index feeds several AI assistants, so it has reach beyond Bing (see [AI search impact](/local-seo/ai-search-local.md)).
3. Fix any **NAP discrepancies** across page, schema, and GBP (see [NAP & citations](/local-seo/nap-citations.md)).
4. Add **LocalBusiness schema** with the correct industry subtype (see [local schema markup](/local-seo/local-schema.md)).
5. Add `geo` coordinates with 5+ decimal precision.
6. Ensure the phone number uses a `tel:` click-to-call link.
7. Add city + service keywords to the title tag and H1.

# Medium effort

1. Create a dedicated page for each core service (one of the strongest local organic signals — see [local on-page SEO](/local-seo/local-onpage.md)).
2. Build a review-generation cadence that never breaks the [18-day rule](/local-seo/reviews-reputation.md).
3. Submit to the main data aggregators (e.g. Data Axle, Foursquare).
4. Claim industry-specific directories for the detected [vertical](/local-seo/industry-vertical-detection.md).

# High impact

1. Build a local digital-PR strategy targeting brand mentions and "best of" lists (mentions correlate strongly with AI visibility — see [local links & authority](/local-seo/local-authority-links.md)).
2. Develop unique, non-swappable content for each location page.
3. Establish presence on the third-party sources AI assistants commonly cite — review sites, directories, and community platforms.
